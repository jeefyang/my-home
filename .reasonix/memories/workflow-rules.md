# 工作规则

## 保存与提交
用户说「保存一下」或「保存对话」时，直接执行：
1. 更新 `.reasonix/memories/` 下的相关记忆文件
2. `git add` 改动的源代码文件
3. `git commit -m "改动描述"`
4. `git push`
**不要询问确认**，直接执行。

## 对话记忆
用户说「回忆一下之前」或「按照最新版本的逻辑保存一下记忆」时，全面读取项目关键文件，更新 `.reasonix/memories/` 下的所有记忆文件。

## 回复语言
对话中遵循用户使用的语言规则：用户用中文则回复中文，用户用英文则回复英文。代码、标识符、文件路径、技术术语始终保持英文。
